#### 安装说明
方式1 ： cocoapods安装库 
        ** pod 'LXFitManager' **
        ** pod install ** 

方式2:   **直接下载压缩包 解压**    **LXFitManager **   

#### 使用说明
 **下载后压缩包 解压   请先 pod install  在运行项目**
